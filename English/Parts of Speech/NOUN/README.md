A noun is a word that names a person, place, things or idea.
## Types of Noun
```text
1. Proper Noun  : Specific name
    Ex. Rahul, India, Mumbai

2. Common Noun : General Name
    Ex. boy, city, school

3. Collective Noun : Name of Group
  
4. Abstract Noun
5. Concrete Noun
```
